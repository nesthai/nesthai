CONSEILS DE TOURNAGE (Smartphone) :
--------------------------------------------------------------------------------

1. PARAMETRES CAMERA :
   - Résolution : minimum 1080p (1920x1080)
   - FPS : 30 images/seconde
   - Format : MP4 (H.264) pour compatibilité maximale
   - Stabilisation : activée si possible

2. CADRAGE :
   - Si portrait (recommandé pour smartphone) : filmer en vertical 9:16
   - Si paysage : filmer en horizontal 16:9
   - Laisser un peu d'espace au-dessus de la tête
   - Cadre moyen (taille à mi-cuisse minimum)

3. AUDIO :
   - Choisir un endroit calme (pas de vent, pas de circulation proche)
   - Parler fort et clairement
   - Éviter les zones avec echo (grandes surfaces vitrées)
   - Test audio : faire un essai de 10 secondes et vérifier

4. LUMIERE :
   - Filmer de jour, lumière naturelle
   - Eviter le contre-jour (soleil derrière la personne)
   - Ombre douce idéale (sous un arbre, balcon couvert)

5. FOND :
   - Contexte Pattaya : mer, piscine, condo, terrasse
   - Éviter les arrière-plans trop chargés ou dérangeants
   - Un plan simple est mieux qu'un plan confus

POST-TRAITEMENT (Si nécessaire) :
--------------------------------------------------------------------------------

1. COMPRESSION (pour réduire la taille) :
   - Utiliser Handbrake (gratuit) : https://handbrake.fr
   - Preset "Web" ou "Fast 1080p30"
   - Format de sortie : MP4
   
2. OUTILS RECOMMANDES :
   - Windows : Clipchamp (intégré à Windows 11) ou DaVinci Resolve (gratuit)
   - Mac : iMovie (gratuit)
   - Mobile : InShot, CapCut

3. ELEMENTS A AJOUTER (optionnel) :
   - Logo Beaulieu en overlay (coin inférieur droit)
   - Titre en début de vidéo : "Marc D. - Lyon, France"
   - Chapitres si vidéo longue (>10min)
